Ruby

Iterations

each: don't care about the indexes. each_ with _index: acts like Hash each_index: just the indexes. Equivalent to "length.times". map: is another way to iterate useful when you want to transform one array into another. select: use when you want to choose a subset inject: generating sums or products, or collecting a single result.

attr

attr_writer :age That gets translated into: def age=(value) @age = value end attr_reader :age That gets translated into: def age @age end attr_accessor :age That gets translated into: def age=(value) @age = value end def age @age end

Comparison

== checks if objects are equal to eachother != not equal to <=> Returns 0 if first equals second 1 if first operand is greater than the second and -1 if first operand is less than the second.

Printing

puts / print call the 'to_s' method on the object AND return nil. Return will be nill Using 'p' calls 'inspect' method on the object. The return statement is used to return values from a Method.
